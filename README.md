# Rock-Paper-Scissors

Rock-Paper-Scissors is a simple game written in JavaScript. This game allows users to play the classic hand game "Rock-Paper-Scissors" against the computer.

## Features

- Play as a human against a computer opponent
- There is an auto play session
- Simple text-based interface for quick and fun gameplay
- Randomized computer moves for unpredictable outcomes
- Score tracking for wins, losses, and ties
- Clean and easy-to-understand source code for learning purposes

## How to Play

1. Run the rock-paper-scissors.html on the browser.
2. Choose one of the options: Rock, Paper, or Scissors.
3. The computer will randomly select its move.
4. The game will display the winner for each round.

## Rules

- **Rock** beats **Scissors**
- **Scissors** beat **Paper**
- **Paper** beats **Rock**
